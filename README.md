# C-FOS Analysis

This repository provides a simple example workflow for analyzing c-Fos immunofluorescence quantification data from rodent neuroscience experiments.

## Overview

The workflow includes:

- loading example c-Fos quantification data
- calculating group-level mean and SEM
- performing basic statistical comparisons
- generating publication-style plots

## Repository Structure

- `data/`: simulated example dataset
- `scripts/`: analysis scripts
- `results/`: output figures generated by the script
